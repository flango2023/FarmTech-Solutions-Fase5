# Como Executar a Entrega 1

O arquivo principal da entrega e o notebook:

- `RichardSchmitz_rm567951_pbl_fase5.ipynb`

Ele ja esta estruturado com:

- celulas de codigo comentadas;
- celulas markdown explicando o raciocinio;
- graficos e tabelas;
- resultados e conclusoes.

## Passos para abrir

1. Instale as dependencias:

```bash
pip install -r requirements.txt
```

2. Abra o notebook no Jupyter:

```bash
jupyter notebook RichardSchmitz_rm567951_pbl_fase5.ipynb
```

ou

```bash
jupyter lab RichardSchmitz_rm567951_pbl_fase5.ipynb
```

## Observacao

Nao e mais necessario copiar e colar codigo manualmente em um novo notebook. A versao oficial da Entrega 1 ja esta pronta no arquivo `.ipynb` do repositorio.
